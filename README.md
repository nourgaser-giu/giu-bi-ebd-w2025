# giu-bi-ebd-w2025
This repository will contain all the codebase referenced in labs for university course Electronic Business Development (EBD - BINF 503) at the German International University (GIU).

## How to run
1. Make sure you have the following software installed
    - Git
    - NodeJS
    - VS Code (Recommended)
2. Clone this repository by running the following command in your terminal.
    ```
    git clone https://github.com/nourgaser-giu/giu-bi-ebd-w2025.git
    ```
3. Open the project in your favorite code editor / IDE. VS Code is recommended.
4. To run any of the JavaScript files in VS Code, open the terminal, then run the file using NodeJS. For example:
    ```
    node lab2/variables.js 
    ```
